This program analyse textual file.
You enter file name and criterion of search using command line.
First argument is file path,and other are the words.
Every word must have prefix '+' or '-'.
Report will print the following sentences:
-all the sentences which have all the words with prefix '+'
-all the sentences which have at least one of the words with prefix '+'
-all the sentences which don't have any of the words with prefix '-'.
Every sentence in the file must end with '.','?' or '!'.
